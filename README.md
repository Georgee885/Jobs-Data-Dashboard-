# Jobs-Data-Dashboard-
Developed an interactive Power BI dashboard to analyze Job Market trends across various roles such as data engineer, data analyst and data scientist. The dashboard includes insights like job distribution by title, geographic job locations and salary averages (hourly and yearly).Integrated datasets and applied DAX measures to visualize trends effectively.

Tools Used:
Power BI, CSV\Excel datasets, DAX, Power Query

Key Insights:
Total Job Count: 479K
Average Yearly Salary: $115K
Average Hourly Rate: $8
